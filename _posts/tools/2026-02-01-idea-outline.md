> 提纲：聚焦快捷键、调试、插件与性能优化，适配 Java 后端开发。

- 基础配置
  - JDK/SDK 配置，编码/换行符，代码风格模板，自动导包与格式化规则。
  - Lombok/Annotation Processing 开启。
- 快捷键与导航
  - 搜索/跳转（双击 Shift、Ctrl+Shift+N/F）、最近文件、书签、拆分窗口。
  - 编辑快捷键：重命名、抽取方法/变量、生成代码、行/块操作。
- 调试与排查
  - 断点类型（行、条件、日志、方法）、Evaluate Expression、Watches。
  - Attach to process / 远程调试；线程/变量视图；内存快照入口。
- 构建与运行
  - Maven/Gradle 导入与配置，运行/调试配置，JVM 参数，Profiles。
- 插件推荐
  - MyBatisX、Rainbow Brackets、IDE Eval Reset（避免违规）、Keymap 适配（VSCode/Emacs）。
- 版本控制集成
  - Git 内置操作、cherry-pick、shelf/stash、变基与冲突解决界面。
- 代码质量
  - Inspection/Alt+Enter 快修，Code Cleanup，格式化/Optimize Imports。
- 性能与内存
  - 启动参数（idea64.vmoptions），禁用不必要插件，索引卡顿排查，离线模式。
- 常见问题
  - 控制台中文乱码，端口占用，Gradle 守护进程，代理配置。
